How It Works

The sticky navigation uses JavaScript to detect which section is currently in the viewport.

When you click a navigation link, it smoothly scrolls to the target section and highlights the clicked link. When you manually scroll, it automatically detects which section you're viewing and updates the active state accordingly.

Try it: Click any link in the navigation above or scroll naturally - the component works both ways!

Frequently Asked Questions

Q: Will this work on mobile devices?

A: Yes! The component is fully responsive. On smaller screens, it collapses into a hamburger menu.

Q: Can I customize the colors?

A: The colors are defined by the design system classes (.blue-background-block and .on-dark). You can override them in your custom CSS if needed.

Q: How do I add more links?

A: Simply add more `` tags in the navigation section, making sure each href matches a section id on your page.

Q: Does it work without jQuery?

A: This version requires jQuery. If you need a vanilla JavaScript version, the logic can be converted.
